# SystemA v.3.1 - System Maintenance & Reporting Tool

## Overview
SystemA v.3.1 is a comprehensive system maintenance, cleaning, and reporting tool designed for Windows, Linux, and macOS. It provides deep cleaning, system optimization, and detailed hardware/software reporting in 9 languages.

## Features
- Multi-language support (TR, EN, DE, FR, ES, PT, RU, ZH, JA)
- Cross-platform support (Windows, Linux, macOS)
- Deep system cleaning and optimization
- Hardware and software inventory reporting
- Network diagnostics and WiFi password recovery
- Email account detection
- System health scoring
- Modern web interface with theme switching

## Installation
1. Clone or download the repository
2. Run `SystemA.bat` (Windows) or `SystemA.sh` (Linux/macOS) as Administrator/Root
3. Follow the on-screen instructions

## Requirements
- Windows 10/11, Linux (Ubuntu/Debian based), or macOS
- Administrator/Root privileges
- Internet connection for updates
