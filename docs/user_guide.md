# SystemA v.3.1 User Guide

## Getting Started
1. Run the program with administrator/root privileges
2. Select your language from the menu
3. Read and accept the disclaimer
4. Choose maintenance operations

## Main Operations

### System Cleaning
- Temporary file cleanup
- Browser cache cleanup
- Windows Update cache cleanup
- Disk cleanup and optimization
- Registry backup (Windows)

### System Updates
- Package manager updates
- Winget updates (Windows)
- Homebrew updates (macOS)
- APT updates (Linux)

### System Restore
- Create system restore points
- Manage shadow copies
- Registry backup

### Network Operations
- DNS flush
- Winsock reset
- Proxy reset
- WiFi password recovery

### Reporting
- System health score (0-100)
- Hardware inventory
- Software inventory
- Network configuration
- Email account detection

## Safety Features
- Full backup recommended before operations
- Confirmation prompts for critical operations
- Restart warnings when needed
- Detailed logging of all operations
